# Blade

AI-powered fintech APIs that transform unstructured financial data into structured, actionable insights.

**Website:** [bladeapi.com](https://bladeapi.com)

## Features

- **Transaction Parser** — Extract structured data from messy bank feeds, SMS alerts, and payment notifications
- **Intelligent Caching** — Known patterns return instantly; new patterns are learned and cached
- **Developer-First** — Clean REST APIs with comprehensive documentation

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production Build

```bash
pnpm build
pnpm start
```
